# Issue Tracker

This repo uses **GitHub Issues** as its issue tracker.

- **Repo**: [XilaiWang/qa-matching](https://github.com/XilaiWang/qa-matching)
- **CLI**: `gh issue` commands (requires `gh` CLI authenticated)
- **Workflow**: Issues are created, triaged, and closed via GitHub Issues

Skills that depend on this config: `to-issues`, `triage`, `to-prd`, `diagnose`.
